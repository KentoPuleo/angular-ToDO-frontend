Willkommen zur ToDo-App

Auf dieser Webapplikation finden sie eine Sign in Page, auf welcher sie ein Konto erstellen können oder sich mit einem bereits erstellten Konto anmelden können.

Nach der anmeldung finden sie eine ToDo-App, auf welcher sie einen Task erstellen, löschen und abhaken können.

Diese Webapplikation wurde für das Modul 322 von Puleo Kento erstellt und er wünscht ihnen viel spass :)
